## 优化目的
使本网站变得更美观更好，从而成为更权威、专业、流量更大的网站，包括：
- 更加自动: 优化scripts下的程序，给网站提供自动更新的功能，不需要人工干预就可以从网上获取最新文章、各种数据；不需要人工干预，你就能完成网站的升级完善。
- 更加专业: 提供中英文两个版本，给英语和中文母语者，围绕LNG贸易领域，提供更加权威和学术的基础知识（包括全球产业情况、全球市场变化、LNG国际贸易情况、LNG交易基础知识、长约合同、现货合同、交易流程、中国LNG情况、中国进口流程等）供初学者入门学习，让用户实时感知最新的行业动态，能收集较专业的资讯。
- 更加开放: 要以英文展示中国情况，针对国外想了解中国LNG和天然气行业的外国LNG交易和研究者，要能提供他们感兴趣的内容比如中国的基础设施、市场情况、买家及与国际资源商合作情况；
- 更加国际化: 要通过中文能让中国人迅速了解全球LNG国际贸易的基本情况，提供国际LNG项目、主要供货商的基本情况和他们主要经济活动合作伙伴和项目。
- 分步骤完善: 根据优化目的分步骤进行优化，正在进行的优化记录在本文件，已经形成的功能写在project.md，过程记录到memory.md。如何还要创建什么文件辅助作业，可用的所有数据源写在datasource-plan.md
