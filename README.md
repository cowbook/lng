# LNG.cool

有关 LNG 贸易与市场的网站（VitePress）。

## 环境

- Node.js 18+
- npm

## 安装

```bash
npm install
```

## 本地开发

```bash
npm run docs:dev
```

## 更新数据

```bash
npm run update:data
```

## 构建

`npm run docs:build` 会先执行数据更新，再执行 VitePress 构建。

```bash
npm run docs:build
```

## 预览

```bash
npm run docs:preview
```
